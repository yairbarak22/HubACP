Frequently Asked Questions

HubACP transforms your WooCommerce store into an AI-ready commerce platform. It provides a secure API that enables ChatGPT to browse your products, create shopping carts, calculate shipping and taxes, and complete purchases - all through natural conversation. It implements OpenAI's Agentic Commerce Protocol (ACP).

Free version: Display up to 2 products in ChatGPT, full checkout integration, complete payment processing, real-time webhook notifications, all security features, setup wizard, professional admin UI, diagnostic tools, and community support (72h response time).

Pro version ($99/year): Everything in Free PLUS unlimited products in ChatGPT feed, priority email support (24h response time), automatic updates, and advanced analytics dashboard (coming soon).

Key difference: The main upgrade benefit is removing the 2-product limit and getting priority support. All core commerce features (checkout, payment, webhooks, security) are included in the Free version!

When a customer tells ChatGPT they want to buy something, ChatGPT uses our secure API to:

  1. Create a checkout session with their items
  2. Calculate shipping costs to their address automatically
  3. Calculate applicable taxes
  4. Process payment securely
  5. Create the order in your WooCommerce store

You get notified via real-time webhook and the order appears in WooCommerce just like a normal order. All communication is secured with HMAC-SHA256 authentication. This feature is available in both Free and Pro versions!

System Requirements: PHP 8.1+, WordPress 6.5+, and WooCommerce 8.0+

PHP Extensions Required: json, hash, curl, mbstring

Setup: You'll also need to register as an OpenAI merchant (free process) and configure the API credentials. Our beautiful setup wizard guides you through this in just 5 minutes.

Extremely secure - we implement enterprise-grade security standards:

  • HMAC-SHA256 authentication - Same security standard used by Stripe and payment platforms
  • Timestamp validation - 5-minute tolerance window prevents replay attacks
  • Rate limiting protection - Prevents API abuse
  • Idempotency key support - Prevents duplicate orders
  • Signature verification - All requests must be cryptographically signed

Every API request is verified before processing to ensure maximum security for your store.
